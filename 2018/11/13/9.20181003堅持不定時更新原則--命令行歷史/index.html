命令行歷史

保存你輸入的命令歷史。可以用它來重複執行命令
登錄Shell時，會讀取命令歷史文件中記錄下的命令~/.bash_history
登錄進Shell后新執行的命令只會記錄在緩存中；這些命令會用戶退出時“追加”至命令歷史文件中

1. 重複前一個命令，有4中方法:

  1. 重複前一個命令使用上方向鍵，并回車執行

  2. 按！！并回車執行

  3. 輸入!-1并回車執行

  4. 按Ctrl+p并回車執行

    2. !:0 執行前一條命令（去除參數）

3. Ctrl+n顯示當前歷史中的下一條命令，但不執行

4. Ctrl+j執行當前命令

5. !n執行history命令輸出對應序號n的命令

6. !-n執行history命令中倒數第n個命令

7. !string 重複前一個以“string”開頭的命令

8. !?string 重複前一個包含string的命令

9. !string:p 僅打印命令歷史，而不執行

10. !$:p 打印輸出!$（上一條命令的最後參數）的內容

11. !:p 打印輸出!(上一條命令中的所有參數)的內容

12. ^string 刪除上一條命令中的第一個string

13. ^string1^string2 將上一條命令中的第一個string1替換為string2

14. !:gs/string1/string2 將上一條命令中所有的string1都替換為string2

15. 使用up（向上）和down（向下）鍵來上下瀏覽從前輸入的命令

16. ctrl+r來在命令歷史中搜索命令

    (reverse-i-search)`’:

17. Ctrl+g:從歷史搜索模式退出

18. 要重新調用前一個命令中最後一個參數

    !$表示
    Esc,.(點擊Esc鍵后鬆開，然後點擊.鍵)
    Alt+.(按住Alt鍵的同時點擊.鍵)

19. command !^ 利用上一個命令的第一個參數做cmd的參數

20. command !$ 利用上一個命令的最後一個參數做cmd的參數

21. command !* 利用上一個命令的全部參數做cmd的參數

22. command !:n 利用上一個命令的第n個參數做cmd的參數

[root@centos7 ~]# echo /etc/sda /etc/sdb
/etc/sda /etc/sdb
[root@centos7 ~]# echo !:2
echo /etc/sdb
/etc/sdb

23. command !n:^ 调用第n条命令的第一个参数

24. command !n:$ 调用第n条命令的最后一个参数

25. command !n:m调用第n条命令的第m个参数

26. command !n:* 调用第n条命令的所有参数

27. command !string:^ 从命令历史中搜索以string 开头的命令，并获取它的第一个参数

28. command !string:$ 从命令历史中搜索以string 开头的命令,并获取它的最后一个参数

29. command !string:n 从命令历史中搜索以string 开头的命令，并获取它的第n个参数

30. command !string:* 从命令历史中搜索以string 开头的命令，并获取它的所有参数
