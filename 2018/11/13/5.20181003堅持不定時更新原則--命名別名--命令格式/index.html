    5.20181003堅持不定時更新原則--命名別名--命令格式

    linux 基础
    创建时间:2018-11-13 11:26
    阅读:
    1. 命名別名
    2. 命令格式
      1. COMMAND [OPTIONS…][ARGUMENTS…]
      2. 注意：
    3. 命令執行：alias–內部–hash表–$PATH–命令找不到

    命名別名

    1.顯示當前Shell進程所有可用的命令別名：

    2.定義別名NAME，其相當于執行命令VALUE
    alias NAME = ‘VALUE’

    3.在命令行中定义的别名，仅对当前shell进程有效
    4.如果想永久有效，要定义在配置文件中
        仅对当前用户：~/.bashrc
        对所有用户有效：/etc/bashrc
    5.编辑配置给出的新配置不会立即生效
    6.bash进程重新读取配置文件

    8.如果别名同原命令同名，如果要执行原命令，可使用
    \ALIASNAME
    “ALIASNAME”
    ‘ALIASNAME’
    command ALIASNAME
    /path/commmand

    命令格式

    COMMAND [OPTIONS…][ARGUMENTS…]

    選項：用於啟用或關閉命令的某個或某些功能
        短選項:-c 例如：-l,-h
        長選項：–word 例如:–all,–human-readable
    參數：命令的作用對象，比如文件名，用戶名等

    注意：

    多個選項以及多參數和命令之間使用空白字符分隔
    取消和結束命令執行：Ctrl+c,Ctrl+d
    多個命令可以用；符號分開

    一個命令可以用\分成多行

    命令執行：alias–內部–hash表–$PATH–命令找不到
